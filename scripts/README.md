# Scripts

Corpus and maintenance scripts live here.

- `mirror_ghalii.py` mirrors GhaLII reference documents.
- `refresh_mirror.py` refreshes the local mirror.
- `check_reference.py` verifies the local reference mirror and resolver.
- `parse_bill.py` parses a bill folder into clause-level JSON.
- `check_bill.py` verifies a parsed bill folder.
- `index_corpus.py` builds local lexical indexes.
- `search_corpus.py` searches local lexical indexes.
- `check_index.py` verifies local lexical indexes.
